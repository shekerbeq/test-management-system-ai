# Пайдаланушы нұсқаулығы

## Оқытушы

1. `register.html` бетінде тіркеледі немесе Google арқылы кіреді.
2. `create-test.html` бетінде тақырып, сұрақ саны, тип, күрделілік, тіл және уақыт лимитін таңдайды.
3. `AI арқылы генерациялау` батырмасы preview сұрақтарын жасайды.
4. Preview ішінде сұрақ мәтінін, жауап нұсқаларын және дұрыс жауаптарды түзетуге болады.
5. Тест сақталған соң `tests.html` бетінде жариялау, жабу, статистика көру және жою әрекеттері бар.
6. Статистика CSV/PDF ретінде жүктеледі.
7. Ашық сұрақ жауаптарын мұғалім қолмен немесе AI бағалау endpoint арқылы тексере алады.

## Студент

1. `dashboard.html` бетінде жарияланған тесттерді көреді.
2. Тестті бастайды, сұрақтарға жауап береді.
3. Уақыт лимиті backend деңгейінде тексеріледі.
4. Тест аяқталған соң жабық сұрақтар бойынша нәтиже бірден көрсетіледі.
5. Бұрын тапсырылған тесттер dashboard ішінде белгіленеді.

## Әкімші

1. `admin-users.html` беті және `/api/admin/*` endpoint-тері арқылы пайдаланушыларды басқарады.
2. Рөлдерді `STUDENT`, `TEACHER`, `ADMIN` мәндеріне өзгерте алады.
3. Audit log және AI usage summary арқылы жүйелік әрекеттерді бақылайды.
